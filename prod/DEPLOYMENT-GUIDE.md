# Workstation Monitor Deployment Guide

## Install on a Workstation

**Required files:**
- `3-Install.bat`
- `Install-Monitor.ps1`
- `WorkstationMonitor.exe`
- `config.json`

**Steps:**
1. Copy the files above to the workstation
2. Double-click `3-Install.bat`
3. Click **Yes** on the admin prompt (UAC)
4. Done - monitor runs automatically for any user who logs in

---

## View the Dashboard

**Required files:**
- `Dashboard.exe`

**Steps:**
1. Double-click `Dashboard.exe`
2. Browser opens to `http://localhost:9090`
3. Click any workstation to see details and graphs

---

## Uninstall

1. Double-click `Uninstall.bat` on the workstation
